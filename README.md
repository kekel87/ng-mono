### Toolings

**NX running**

```bash
nx [CMD] # Global install  npm install -g nx
npx nx [CMD]
npm run nx [CMD]
npm run ng [CMD]
```

**Runnning**

```bash
npm start # netatmo default
npm start [app-name]
```

**Code formating**

```bash
# nx config format
nx format

# apps format
nx lint [app-lib-name]
nx affected:lint --fix
nx run-many --target=lint --fix
```

**Code linting**

```bash
# nx config check
nx format:check

# apps lint
nx lint [app-lib-name]
nx affected:lint
nx run-many --target=lint
```

**Unit testing**

```bash
nx test [app-lib-name]
nx affected:test
nx run-many --target=test
```

**End to end testing**

```bash
nx e2e [app-name-e2e]

nx affected:e2e
nx run-many --target=e2e
```

**NX command**

```bash
nx graph
nx affected:graph 😍
nx print-affected --type=app --select=projects
nx workspace-lint
```
